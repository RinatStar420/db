API - программный интерфейс приложения - это набор функция для доступа к какой либо услуги.
DB-API - стандартный API в Python, предназначенный для предоставления доступа к реляционным базам данных.

Рассмотрим его функции:

connect() - создание соединения с бд.
cursor() - создание объекта курсора для работы с запросами
execute() и executemany() - запуск одной и более команд SQL
fethcone(), fetchmany(), fetchall() - получение результата работы execute().
