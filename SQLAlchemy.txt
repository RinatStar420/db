Самая популярная библиотека для работы с разными бд - SQLAlchemy.
Можно использовать SQLAlchemy на нескольких уровнях:

- На самом низком уровне она работает с пулами соединений к базе данных, выполняет команды SQL  и возвращает результат

- Следующий уровень - язык выражений SQL, построитель SQL  в Python

- Самый высокий уровень - это слой ORM (Object Relation Model) который использует язык выражений SQL Expression и
связывает код приложения с реляционными структурами данных.


Не нужно устанавливать драйверы других бд - он будет определен сам с помощью строки соединения, которую предоставляет
SQLAlchemy.
Строка выглядит примерно так: dialect + driver :// user : password @ host : port / dbname

В эту строку нужно поместить следующие значения:

- dialect - тип бд
- driver - драйвер, которые мы хотим использовать для бд
- user и password  - строки аутификации для этой бд
- host и port - расположение сервера бд (значение порта нужно использовать только при нестандартном порте)
- dbname - имя бд, куда нужно подключиться
